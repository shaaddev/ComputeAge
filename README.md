## 🌎 Set up 🛠
#
I. Install the dependencies

```bash
$ npm install
or 
$ bun install
```

II. Start the development server

```bash
$ npm run dev
or 
$ bun run dev
```

## 🛫 Build and run production ⚙️
#

```bash
$ npm run build
or 
$ bun run build
```
